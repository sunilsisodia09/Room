#  RoomFinder App

A full-stack MERN application for finding and listing PGs/rooms with authentication, provider dashboard, and image upload support.

---

#  Features

##  User Side
- User Signup / Login
- Search PG / Rooms
- View Room Details
- Responsive UI

##  Provider Side
- Provider Signup / Login
- Add Listings
- Manage Listings
- Upload Images

---

#  Tech Stack

## Frontend
- React.js
- Vite
- React Router DOM
- Axios
- CSS

## Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- Multer
- CORS
